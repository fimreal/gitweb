gitweb 是什么

把任意 Git 仓库变成网页，直接在浏览器里看——不用克隆、不用构建。适合用 git 管理的笔记、文档、wiki 或任何内容。

  1. 粘贴一个 Git 仓库地址（GitHub / GitLab / Gitea），点创建。
  2. 打开生成的链接，仓库里的文件就渲染成可浏览的网页。
  3. 用浮动的文件树在文件间跳转，Markdown 和代码都会自动渲染。
  4. 私有仓库？打开后点 🔑 凭据 输入 Token 或账号密码。
